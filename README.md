# どうやってやるか?

実行委員とボランティアによってソーシャル翻訳します。

- あまり正確でなくてよい
- 読みやすい日本語で
- アジャイル初心者セミナー(11/27 Fri)に第一版を印刷して配りたいので、10末を一旦の締め切りに
- 各自、t フォルダにあるファイルから、好きなものを取って、英語を残したままで編集して、直接Commit してください。たぶん、ぶつからないですし、ぶつかっても大丈夫です。
